## 五、设计师编辑指南

### 5.1 修改技能效果 (GE)
1. 打开对应的 `DBAGE_{Zodiac}_{Skill}.cpp` 文件
2. 在构造函数中添加效果逻辑：
   - `DurationPolicy`: 持续时间类型 (Instant/Infinite/HasDuration)
   - `Modifiers`: 属性修饰符列表
   - 伤害、治疗、增益、减益等

### 5.2 修改视觉特效 (Cue)
1. 打开对应的 `DBACue_{Zodiac}_{Skill}.cpp` 文件
2. 在 `OnExecuteGameplayCue()` 中实现特效播放
3. 在 `OnActiveGameplayCue()` 中实现持续特效
4. 在 `OnRemoveGameplayCue()` 中实现特效移除

### 5.3 修改技能行为 (GA)
1. 打开对应的 `DBAGameplayAbility_{Zodiac}_{Skill}.cpp` 文件
2. 重写 `ActivateAbility()` 实现技能逻辑
3. 使用 `CommitAbilityCost()` 消耗能量
4. 使用 `EndAbility()` 结束技能

### 5.4 使用脚本重新生成
如需批量修改模板，编辑 `Scripts/generate_zodiac_abilities.py` 或 `Scripts/generate_gas_cues_and_effects.py`，然后运行：
```bash
python Scripts/generate_zodiac_abilities.py
python Scripts/generate_gas_cues_and_effects.py
```
